Recomendação de portas:
  - Servidor: 8080 (já configurado corretamente)
  - Cliente Flutter: 3000 (consistente, boa para desenvolvimento)
  - Alternativa: Pode usar qualquer porta livre (5000, 8000, etc.)

  Comandos recomendados:
  # Servidor (sempre na mesma porta)
  cd server && dart pub global run dart_frog_cli:dart_frog dev

  # Cliente (sempre na porta 3000 para consistência)
  cd client && flutter run -d chrome --web-port 3000
